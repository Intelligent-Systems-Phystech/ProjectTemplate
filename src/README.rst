************
Installation
************

Requirements
============

- Python 3.*
- pip 20.0.2

Installing by using PyPi
========================

Install
-------
.. code-block:: bash

	git clone https://github.com/Intelligent-Systems-Phystech/ProjectTemplate.git
	cd ProjectTemplate
	python3 -m pip install ./src/

Uninstall
---------
.. code-block:: bash

  python3 -m pip uninstall mylib
